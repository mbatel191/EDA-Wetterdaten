# EDA Wetterdaten Deutschland (1996-2021)
Dies ist eine explorative Datenanalyse (EDA) eines Datensatzes vom Deutschen Wetterdienst (DWD). Der Datensatz enthält gemessene Temperaturen von 1996 bis 2021.

## Projektziel
Explorative Datenanalyse von Temperaturmessungen in Deutschland von 55 Wetterstationen im Zeitraum 1996-2021 (10-Minuten-Intervalle).
In diesem Projekt analysiere ich ein öffentlich verfügbares Datenset, um:
- Zusammenhänge zu erkennen
- Daten zu bereinigen
- erste Erkenntnisse zu gewinnen
Ziel war es auch, praktische Erfahrungen mit Python und Datenanalyse zu sammeln.

## Datenquelle
- Datei: "german_temperature_data_1996_2021_from_selected_weather_stations.csv"
- Zeitraum: 01.01.1996-31.12.2021
- Messintervall: alle 10 Minuten
- Gemessen von: Deutscher Wetterdienst (DWD)
- Lizenz: Creative Commons BY 4.0 "CC BY 4.0"

## Beschreibung
- Dieser Datensatz enthält Temperaturmessungen von 55 Wetterstationen in Deutschland in 10-Minuten-Abständen.

## Ordnerstruktur
- 'data/' – Erfordert den Originaldatensatz (https://www.kaggle.com/datasets/alioraji/weather-data-nov-2015)
- 'analysis/' – Jupyter Notebook mit Analyse
